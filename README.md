# supnex_test_project

## Installation

```bash
git clone git@github.com:smrrazavian/supnex_test_project.git && cd supnex_test_project
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## Running the app with docker-compose

```bash
# development
docker-compose up -d --build
```
